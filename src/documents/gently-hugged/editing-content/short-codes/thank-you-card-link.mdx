# Thank You Card Link

```
<ThankYouCardLink />
```
OR
```
<ThankYouCardLink>
    text to display here
</ThankYouCardLink>
```
Display a link to download the donation thank you card. You can specify the text to show on the web site as shown in the second example. If you omit this the title will be used as the text. The PDF document can be found in the [Downloads section](/gently-hugged/contents/downloads/thank-you-card/).
