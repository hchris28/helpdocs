# Handmade Guidelines Link

```
<HandmadeGuidelinesLink />
```
OR
```
<HandmadeGuidelinesLink>
    text to display here
</HandmadeGuidelinesLink>
```
Display a link to the handmade guidelines document. You can specify the text to show on the web site as shown in the second example. If you omit this the title will be used as the text. 

The document can be found in the [Downloads section](/gently-hugged/contents/downloads/handmade-guidlelines-document/).
