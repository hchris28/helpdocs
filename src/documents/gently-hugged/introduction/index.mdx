# Introduction

## What is this?

[The Gently Hugged web site](https://gentlyhugged.org/) can be administered using a content management system. This document aims to provide all the information you will need to make changes such as updating text or adding new images.

## How to use this document

You can browse topics using the menu on the left. If you're having an issue you can search by keyword using the search bar at the top of the page.

## Logging in to the Content Management System (CMS)

The CMS, a.k.a. the admin interface, can be found at this address: [https://gentlyhugged.org/admin/](https://gently-hugged.netlify.app/admin/). You can log in using the username and password provided to you.

## Nerdy stuff

The CMS we're using is called [Decap](https://decapcms.org/), it's powered by [Netlify](https://www.netlify.com/). The web site itself is written in React using [Gatsby](https://www.gatsbyjs.com/) to generate the HTML pages.
