The distribution sites list is a list of distribution sites with images. Each distribution site has the following fields:

* **NAME**\
    The name of the distribution site.

* **IMAGE** (Optional)\
    The logo of the distribution site. If a logo is not supplied, the Name is used instead.  

* **LABEL** (Optional)\
    A label that shows under the logo or the name.

<Aside>
    You can add the distribution siteshort code list to a page using the [`<DistributionSites />` short code](/gently-hugged/editing-content/short-codes/distribution-sites/). 
</Aside>
