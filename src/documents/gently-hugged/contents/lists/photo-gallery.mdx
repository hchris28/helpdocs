# Photo Gallery

The photo gallery is a set pf photos that can be displayed on the web site like slide show. Each photo has the following fields:

* **CAPTION**\
    The name of the Distribution Site.

* **IMAGE** (Optional)\
    The image to show. There are no hard and fast rules for image dimensions, they will be resized to fit the screen as needed.
<Aside>
    You can add the photo gallery to a page using the [`<PhotoGallery />` short code](/gently-hugged/editing-content/short-codes/photo-gallery/). 
</Aside>
