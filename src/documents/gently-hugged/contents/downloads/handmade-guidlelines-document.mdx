# Handmade Guidelines Document

The Handmade Guidelines Document is for sharing guidelines for making items to donate.

<Aside>
    A link to the Handmade Guidelines Document can be added to a page using the [`<HandmadeGuidelinesLink />` Short Code](/gently-hugged/editing-content/short-codes/handmade-guidelines/).
</Aside>

* **TITLE**\
    This is the title of the document. This will be used as the link text unless you specify different text to use in the Short Code.

* **FILE**\
    The File to make available for download.

* **DESCRIPTION** (Optional)\
    The Description is optional. It is not displayed on the web site at this time.
