# Patterns

This section allows you to manage the patterns that visitors can download. You can add new files, remove old ones, change the title associated with the file, and re-order the list of patterns

<Aside>
    You can add a list of patterns to a page using the [`<PatternLinks />` short code](/gently-hugged/editing-content/short-codes/pattern-links/). 
</Aside>

* **TITLE**\
    This is the title of the document. This will be used as the link text.

* **FILE**\
    The file to make available for download. Use a PDF for best results.

* **DESCRIPTION** (Optional)\
    The description is optional. It is not displayed on the web site at this time.
