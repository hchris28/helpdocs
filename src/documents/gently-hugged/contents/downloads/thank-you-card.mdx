# Thank You Card

The thank you card is a card that visitors can download after making a donation to let someone know a donation was made in theor honor.

<Aside>
    A link to the thank you card document can be added to a page by using the [`<ThankYouCardLink />` short code](/gently-hugged/editing-content/short-codes/thank-you-card-link/). 
</Aside>

* **TITLE**\
    This is the title of the document. This will be used as the link text unless you specify different text to use in the Short Code.

* **FILE**\
    The file to make available for download. Use a PDF for best results.

* **DESCRIPTION** (Optional)\
    The description is optional. It is not displayed on the web site at this time.
