This is default view when you first log in. This is where you will find all the options to make changes to content on the web site.

On the left side you will see a navigation menu labeled **Collections** containing links to different sections where you can edit various peices of web site content. 

![The collections menu](/images/collections-menu.jpg)

The menu on the left also contains a search field where you can search for items containing keywords.
